# File System Tools

File system tools enable agents to interact with files within their sandboxed environments. All operations are scoped to the session directory for security.

## Core File Operations

### view_file

Read the content of a file within the session sandbox.

<ParamField path="path" type="string" required>
  Path to the file relative to session root
</ParamField>

<ParamField path="workspace_id" type="string" required>
  Workspace identifier
</ParamField>

<ParamField path="agent_id" type="string" required>
  Agent identifier
</ParamField>

<ParamField path="session_id" type="string" required>
  Session identifier
</ParamField>

<ParamField path="encoding" type="string" default="utf-8">
  File encoding
</ParamField>

<ParamField path="max_size" type="integer" default="10485760">
  Maximum file size in bytes (default: 10MB)
</ParamField>

<CodeGroup>

```python Example
result = view_file(
    path="data/config.json",
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)

if result["success"]:
    content = result["content"]
    print(f"File size: {result['size_bytes']} bytes")
    print(f"Lines: {result['lines']}")
```

```json Response
{
  "success": true,
  "path": "data/config.json",
  "content": "{\"key\": \"value\"}",
  "size_bytes": 16,
  "lines": 1
}
```

</CodeGroup>

### write_to_file

Create a new file or append content to an existing file.

<ParamField path="path" type="string" required>
  Path to the file relative to session root
</ParamField>

<ParamField path="content" type="string" required>
  Content to write to the file
</ParamField>

<ParamField path="workspace_id" type="string" required>
  Workspace identifier
</ParamField>

<ParamField path="agent_id" type="string" required>
  Agent identifier
</ParamField>

<ParamField path="session_id" type="string" required>
  Session identifier
</ParamField>

<ParamField path="append" type="boolean" default="false">
  Whether to append to the file instead of overwriting
</ParamField>

<CodeGroup>

```python Create File
result = write_to_file(
    path="output/results.json",
    content=json.dumps(data, indent=2),
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)
```

```python Append to File
result = write_to_file(
    path="logs/events.log",
    content="New event logged\n",
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789",
    append=True
)
```

</CodeGroup>

### list_dir

List directory contents with file metadata.

```python
result = list_dir(
    path="data",
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)

for item in result["items"]:

```

## Search Operations

### grep_search

Search file contents using regex patterns.

<ParamField path="pattern" type="string" required>
  Regex pattern to search for
</ParamField>

<ParamField path="path" type="string" required>
  Directory or file to search
</ParamField>

<ParamField path="workspace_id" type="string" required>
  Workspace identifier
</ParamField>

<ParamField path="agent_id" type="string" required>
  Agent identifier
</ParamField>

<ParamField path="session_id" type="string" required>
  Session identifier
</ParamField>

```python Example
result = grep_search(
    pattern="error|warning",
    path="logs/",
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)

for match in result["matches"]:
    print(f"{match['file']}:{match['line']}: {match['text']}")
```

## File Modification

### replace_file_content

Replace text in a file using string substitution.

```python
result = replace_file_content(
    path="config.yaml",
    old_text="debug: false",
    new_text="debug: true",
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)
```

### apply_diff

Apply a diff patch to a file.

```python
diff_content = '''
--- a/file.txt
+++ b/file.txt
@@ -1,3 +1,3 @@
 line 1
-line 2
+line 2 modified
 line 3
'''

result = apply_diff(
    path="file.txt",
    diff=diff_content,
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)
```

### apply_patch

Apply a unified patch file.

```python
result = apply_patch(
    path="src/",
    patch_content=patch_data,
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)
```

## Command Execution

### execute_command_tool

Execute shell commands within the sandbox.

<Warning>
This tool executes arbitrary shell commands. Use with caution and proper input validation.
</Warning>

```python
result = execute_command_tool(
    command="ls -la",
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789",
    timeout=30
)

if result["success"]:
    print(result["stdout"])
    print(f"Exit code: {result['exit_code']}")
```

## Data Management

### save_data

Save large data to disk for later retrieval.

<ParamField path="filename" type="string" required>
  Simple filename like 'results.json' (no paths)
</ParamField>

<ParamField path="data" type="string" required>
  String data to write (typically JSON)
</ParamField>

<ParamField path="data_dir" type="string" required>
  Absolute path to the data directory
</ParamField>

```python
import json

data = {"users": [...], "total": 1000}
result = save_data(
    filename="users.json",
    data=json.dumps(data, indent=2),
    data_dir="/workspace/data"
)

print(f"Saved {result['size_bytes']} bytes to {result['filename']}")
```

### load_data

Load previously saved data with pagination.

<ParamField path="filename" type="string" required>
  Filename to load
</ParamField>

<ParamField path="data_dir" type="string" required>
  Absolute path to the data directory
</ParamField>

<ParamField path="offset_bytes" type="integer" default="0">
  Byte offset to start reading from
</ParamField>

<ParamField path="limit_bytes" type="integer" default="10000">
  Maximum bytes to return (default: 10KB)
</ParamField>

<CodeGroup>

```python First Page
result = load_data(
    filename="users.json",
    data_dir="/workspace/data"
)

data = json.loads(result["content"])
print(f"Loaded {result['bytes_read']} bytes")

if result["has_more"]:
    next_offset = result["next_offset_bytes"]
```

```python Next Page
result = load_data(
    filename="users.json",
    data_dir="/workspace/data",
    offset_bytes=10000
)
```

</CodeGroup>

## CSV Operations

### csv_read

Read rows from a CSV file.

```python
result = csv_read(
    path="data/users.csv",
    limit=100,
    offset=0
)

for row in result["rows"]:
    print(row)
```

### csv_write

Write a new CSV file.

```python
headers = ["name", "email", "age"]
rows = [
    ["Alice", "alice@example.com", 30],
    ["Bob", "bob@example.com", 25]
]

result = csv_write(
    path="output/users.csv",
    headers=headers,
    rows=rows
)
```

### csv_append

Append rows to an existing CSV file.

```python
new_rows = [
    ["Charlie", "charlie@example.com", 35]
]

result = csv_append(
    path="output/users.csv",
    rows=new_rows
)
```

### csv_sql

Query CSV files using SQL (DuckDB).

```python
result = csv_sql(
    path="data/sales.csv",
    query="SELECT region, SUM(amount) FROM sales GROUP BY region"
)

for row in result["rows"]:
    print(f"{row['region']}: ${row['SUM(amount)']}")
```

## Excel Operations

### excel_read

Read rows from an Excel sheet.

```python
result = excel_read(
    path="data/report.xlsx",
    sheet="Sales",
    limit=100
)
```

### excel_write

Write a new Excel file.

```python
data = {
    "Sheet1": {
        "headers": ["Name", "Value"],
        "rows": [["A", 1], ["B", 2]]
    }
}

result = excel_write(
    path="output/report.xlsx",
    data=data
)
```

### excel_sql

Query Excel sheets using SQL (DuckDB).

```python
result = excel_sql(
    path="data/report.xlsx",
    query="SELECT * FROM Sheet1 WHERE Value > 10"
)
```

## PDF Operations

### pdf_read

Extract text from PDF files.

```python
result = pdf_read(path="documents/report.pdf")

for page in result["pages"]:

```

## Security

<Note>
All file operations are sandboxed to the session directory. Path traversal attempts are blocked.
</Note>

### Path Security

```python
from aden_tools.tools.file_system_toolkits.security import get_secure_path

# Safe path resolution
secure_path = get_secure_path(
    path="data/file.txt",
    workspace_id="ws_123",
    agent_id="agent_456",
    session_id="session_789"
)
# Returns: /workspace/ws_123/agent_456/session_789/data/file.txt

# Blocked attempts
get_secure_path("../../etc/passwd", ...)  # Blocked
get_secure_path("/etc/passwd", ...)       # Blocked
```

## Best Practices

<AccordionGroup>
  <Accordion title="Always Read Before Modifying">
    Use `view_file` to inspect content before applying changes with `replace_file_content` or `apply_diff`.
  </Accordion>

  <Accordion title="Use Data Tools for Large Results">
    When tool results exceed context limits, use `save_data` to write to disk and `load_data` to retrieve with pagination.
  </Accordion>

  <Accordion title="Validate File Paths">
    Always validate paths are within the expected directory structure before operations.
  </Accordion>

  <Accordion title="Handle Encoding Errors">
    Specify the correct encoding parameter when reading non-UTF-8 files.
  </Accordion>
</AccordionGroup>
